# Tab Master - UI 设计规范 v2.0

## 一、设计概览

### 整体设计风格
**「轻盈现代 + 毛玻璃质感」**
- 大面积留白，呼吸感强
- 毛玻璃（glassmorphism）效果增加层次
- 柔和渐变，轻盈阴影
- 圆角色彩，视觉舒适
- 优雅动效，流畅自然

**用途**: 页面顶部，显示用户最常访问的网站快捷方式，模仿 Chrome 默认新标签页

**尺寸规范**:
- 宽度: 100%
- 高度: 72px
- 内边距: 0 16px
- 图标尺寸: 48px × 48px（可点击区域64px包含间距）

**样式**:

- 圆角: 0（与页面顶部对齐）
- 下边框: 1px solid rgba(255,255,255,0.5)

**布局**:
```
┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  🔍  ───────────────────── 搜索框 ─────────────────────  ⭐  ⚙️        │
│                                                                         │
│  ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐     │
│  │ 🌐   │ │ 📧   │ │ 📅   │ │ 📁   │ │ 🎬   │ │ 🛒   │ │ ➕   │     │
│  │ Google│ │ 邮箱  │ │ 日历  │ │ 云盘  │ │ 视频  │ │ 购物  │ │ 添加  │     │
│  └──────┘ └──────┘ └──────┘ └──────┘ └──────┘ └──────┘ └──────┘     │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

**快捷链接样式**:
- 图标: 48px × 48px，居中
- 背景: 根据网站主色调或渐变色
- 圆角: 12px
- hover: scale(1.08) + 阴影加深
- 标题: 13px，居中于图标下方8px

**状态**:

| 状态 | 样式描述 |
|------|----------|
| 默认态 | 白色图标背景，柔和阴影 |
| Hover态 | scale(1.08)，阴影加深，轻微上浮 |
| 点击态 | scale(0.95)，快速回弹 |
| 加载态 | 图标显示骨架屏动画 |

**数据绑定**:
- 数据来源: chrome.topSites.get() 获取最常访问网站
- 显示: favicon + 标题（最多8个）
- 自定义: 用户可添加/删除快捷链接（存储在 chrome.storage.local）

**交互行为**:
- onClick: 打开对应网站（chrome.tabs.create）
- onContextMenu: 右键菜单（删除、编辑）
- onDrag: 可拖拽调整顺序

## 四、快捷访问功能详解

### 4.1 数据来源
```javascript
// 使用 Chrome topSites API 获取最常访问
chrome.topSites.get(topSites => {
  // 返回最常访问的网站列表
  // [{title: 'Google', url: 'https://google.com', favicon: '...'}, ...]
});
```

### 4.2 默认快捷链接
- 首次安装时显示预设快捷链接（用户可自定义）
- 预设: Google搜索、Gmail、日历、云盘、YouTube、购物 等

### 4.3 用户自定义
- 右键快捷链接 → 删除
- 拖拽调整顺序
- 点击"+"添加自定义链接
- 存储在 chrome.storage.local

### 4.4 快捷搜索
- 搜索框支持切换搜索引擎
- 点击切换图标选择: Google / 百度 / 必应
- 快捷键: Ctrl/Cmd + K 聚焦搜索框

## 六、技术实现备注

### 性能优化
- 毛玻璃效果使用 `will-change: transform` 优化
- 图片使用 lazy loading
- 骨架屏使用 CSS 动画，避免 JS 动画

### Chrome API
```javascript
// 获取最常访问
chrome.topSites.get(callback)

// 快捷链接存储
chrome.storage.local.get(['shortcuts'])
chrome.storage.local.set({ shortcuts: [...] })
```

### 响应式断点 v2.0

| 断点 | Tab卡片列数 | 快捷链接数 |
|------|-------------|------------|
| 超大 >1400px | 7列 | 8个 |
| 大 1100-1400px | 6列 | 7个 |
| 中 800-1100px | 5列 | 6个 |
| 小 <800px | 4列 | 5个 |
