## 一、测试环境

### 1.1 环境要求

| 项目 | 要求 |
|------|------|
| 操作系统 | macOS / Windows / Linux |
| Chrome版本 | Chrome 88+ |
| Node.js | v18+ |
| npm | v8+ |

### 1.2 准备步骤

```bash
# 1. 进入项目目录
cd tab-master/frontend

# 2. 安装依赖
npm install

# 3. 构建扩展
npm run build

# 4. 加载到Chrome
# - 打开 chrome://extensions/
# - 开启"开发者模式"
# - 点击"加载已解压的扩展程序"
# - 选择 frontend/dist 目录

# 5. 安装Playwright（如未安装）
npm install -D @playwright/test
npx playwright install chromium

# 6. 运行测试
npx playwright test tests/tab-master.spec.js
```

## 三、手动测试清单

由于当前环境限制，请按以下清单手动测试：

### 3.1 基础功能测试

- [ ] 打开新标签页，确认显示Tab Master界面
- [ ] 确认有Tab列表展示
- [ ] 确认分组自动折叠

### 3.2 搜索功能测试

- [ ] 在搜索框输入关键词，验证实时过滤
