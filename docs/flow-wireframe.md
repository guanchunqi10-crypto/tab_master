## 2. 组件说明

### 2.1 搜索栏 (SearchBar)
- **位置**: 页面顶部，固定定位
- **尺寸**: 宽度100%，高度48px，内边距16px
- **样式**: 圆角8px，浅灰色背景 #F1F3F4，hover变深
- **图标**: 左侧搜索图标，右侧清除按钮（有内容时显示）
- **交互**: 输入时实时搜索，防抖300ms

### 2.2 Tab卡片 (TabCard)
- **尺寸**: 宽度160px，高度auto，最小高度100px
- **样式**: 白色背景，圆角8px，轻微阴影
- **内容布局**:
  ```
  ┌────────────────┐
  │ 📄 [favicon]   │  ← 32x32 favicon
  │                │
  │ 页面标题...     │  ← 最多2行，超出省略
  │ example.com    │  ← 域名，灰色小字
  │          [✕]   │  ← 关闭按钮，hover显示
  └────────────────┘
  ```
- **状态**:
  - Default: 白色背景
  - Hover: 背景 #F8F9FA，显示关闭按钮
  - Dragging: 透明度0.7，轻微放大

### 2.3 分组标题 (GroupHeader)
- **样式**: 字体粗体16px，左侧折叠箭头 + 分组名称 + Tab数量
- **交互**:
  - 点击箭头: 展开/折叠分组
  - 右键: 显示操作菜单
- **展开状态**: 箭头向下，显示内部Tab卡片
- **折叠状态**: 箭头向右，仅显示标题

### 2.4 底部工具栏 (BottomBar)
- **位置**: 页面底部，固定定位
- **高度**: 120px
- **分区**:
  - 左侧 50%: 最近关闭
  - 右侧 50%: 最常访问
- **Tab展示**: 横向滚动，Tab卡片尺寸略小(120px宽)

### 2.5 右键菜单 (ContextMenu)
- **触发**: 右键Tab卡片或分组标题
- **Tab卡片菜单**:
  - 跳转到该Tab
  - 关闭该Tab
  - 移动到分组 → [子菜单：各分组]
  - 复制链接
  - 固定/取消固定
- **分组标题菜单**:
  - 展开/折叠
  - 重命名
  - 删除分组
  - 合并到... → [子菜单]

## 3. 交互细节

### 3.1 搜索交互
1. 用户聚焦搜索框 → 显示边框高亮
2. 输入内容 → 实时过滤，隐藏不匹配的Tab和分组
3. 无匹配时 → 显示空态插图 + "没有找到匹配的Tab"
4. 清空内容 → 恢复全量显示

### 3.2 拖拽交互
1. 鼠标按下Tab卡片 → 进入拖拽模式
2. 拖动过程中 → 卡片跟随鼠标，其他Tab让出位置
3. 拖到分组上方 → 分组高亮提示"可放置"
4. 释放 → Tab移动到新分组，保存状态
5. 取消拖拽（按ESC或拖出页面） → 卡片回到原位

### 3.3 关闭Tab交互
1. Hover Tab卡片 → 显示关闭按钮
2. 点击关闭按钮 → 卡片滑出消失，添加一条到最近关闭
3. 关闭动画: 200ms fadeOut + slideLeft
