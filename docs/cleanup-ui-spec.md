# Tab Master 智能清理助手 UI 设计规范 v2.0

> 设计理念：让标签页清理变成一种「爽」的体验，把"不敢关"变成"想关"。

---

## 一、设计概览

### 整体设计风格
**风格定位**：Neo-Morphism（新一代玻璃拟态）+ 游戏化交互
- 参考：iOS 控制中心 + Apple Watch 活动环 + 游戏成就系统
- 关键词：流畅、满足、即时反馈、游戏化

### 配色方案
```
主色调：
- Primary: #6366F1 (靛蓝紫)
- Primary Light: #818CF8
- Primary Dark: #4F46E5

功能色：
- 建议关闭: #F59E0B (琥珀橙) → #EF4444 (警示红，紧急)
- 今天会看: #10B981 (翡翠绿)
- 以后可能: #3B82F6 (天空蓝)
- 随手开的: #8B5CF6 (紫罗兰)

背景系统：
- 渐变起点: #E0E7FF (淡紫)
- 渐变中点: #C7D2FE
- 渐变终点: #A5B4FC
- 毛玻璃: rgba(255, 255, 255, 0.7) + backdrop-blur

文字：
- 主文字: #1E1B4B (深紫黑)
- 次文字: #6B7280 (灰色)
- 标签文字: #FFFFFF
```

### 字体规范
```
标题: Inter Bold, 18px
副标题: Inter Semibold, 14px
正文: Inter Medium, 13px
标签/徽章: Inter Bold, 10px
```

### 间距规范
```
基础单位: 4px
常用间距: 8px / 12px / 16px / 24px
卡片内边距: 16px
网格间距: 12px
圆角: 12px (卡片) / 16px (大卡片) / 9999px (标签)
```

### 动效规范
```
基础时长: 200ms (快速) / 300ms (标准) / 500ms (强调)
缓动函数: cubic-bezier(0.4, 0, 0.2, 1)
弹跳效果: cubic-bezier(0.68, -0.55, 0.265, 1.55)

关键动效：
- 关闭动画: 缩放 1→0.8 + 透明度 1→0 + 向上飘散
- 评分徽章: 脉冲呼吸动画 (scale 1→1.1→1, 2s循环)
- 一键清理: 卡片依次飞出 + 成功粒子效果
- 分类切换: 平滑滑动 + 数量跳动
```

## 二、组件规范

### 1. 分类筛选标签组 (CategoryFilter)

**外观**：
- 横向排列 4 个标签，胶囊形状
- 激活态：渐变填充 + 白色文字 + 微妙阴影
- 未激活态：白色半透明背景 + 灰色文字
- 每个标签右侧显示数量徽章

**状态列表**：
```
- 默认态: 白色/70%透明，文字#6B7280
- Hover态: 背景变亮，文字变深，轻微上浮translateY(-2px)
- 激活态: 渐变背景#6366F1→#8B5CF6，文字白色，阴影加深
- 按下态: scale(0.95)，阴影收缩
```

**交互行为**：
- 点击：切换分类，标签组整体轻微弹跳
- 数量变化：数字从右滑入动画

**规格**：
```
高度: 36px
内边距: 16px 水平
圆角: 9999px (全圆角)
间距: 8px
数量徽章: 右侧 4px，圆形，背景继承，文字变小
```

### 2. 清理建议卡片 (CleanupSuggestionCard)

**外观**：
- 大圆角卡片，渐变边框
- 左侧：图标区域（带脉冲动画）
- 右侧：文案 + 操作按钮
- 底部：进度条（如果有批量操作）

**状态列表**：
```
- 默认态: 渐变边框，背景白色/80%透明
- 有建议态: 边框渐变橙色→红色，图标闪烁
- 无建议态: 边框变淡，图标灰显
- 操作中态: 进度条动画，按钮显示进度
- 完成态: 边框变绿，图标打勾，1.5s后淡出
```

**交互行为**：
- 卡片入场：从下方滑入 + 弹性放大
- 一键清理：点击后，卡片开始执行动画
- 数字变化：参与数字的卡片会轻微震动

**规格**：
```
最小高度: 100px
圆角: 20px
边框: 2px 渐变
内边距: 20px
背景: 白色/80%透明 + backdrop-blur(12px)
```

**子元素**：

#### 2.1 智能图标 (SmartIcon)
```
外观: 圆形容器，内有灯泡/魔法棒图标
动画:
  - 默认: 缓慢脉冲 scale(1→1.05, 2s)
  - 有建议: 快速脉冲 + 金色光晕
  - 点击: 爆发光效
尺寸: 48px × 48px
```

#### 2.2 统计数据展示 (StatsDisplay)
```
外观: 三个数字垂直排列
动画:
  - 数字变化时从0滚动到目标值
  - 单位标签淡入
间距: 每个统计 8px
```

#### 2.3 快速操作按钮 (QuickActionButton)
```
外观: 圆角矩形，渐变填充
动画:
  - Hover: 光泽扫过动画
  - 按下: 轻微下沉
  - 加载: 边框进度动画
尺寸: 高度 40px，内边距 20px
```

### 3. 可关闭标识徽章 (CloseableBadge)

**外观**：
- 胶囊形状，橙色/红色渐变
- 图标 + 文字
- 右上角有关闭叉号（暗示可一键关闭）

**状态列表**：
```
- 低优先级(60-70分): 橙色 #F59E0B，文字"可关闭"
- 中优先级(70-85分): 橙红渐变，文字"建议关闭"
- 高优先级(85-100分): 红色 #EF4444，文字"立刻关闭"，脉冲动画
```

**交互行为**：
- Hover：轻微放大 + 显示完整提示
- 点击：触发快速关闭，卡片飞出
- 入场：从左上角弹入

**动画细节**：
```css
/* 脉冲动画 - 高优先级 */
@keyframes pulse-glow {
  0%, 100% {
    box-shadow: 0 0 0 0 rgba(239, 68, 68, 0.4);
    transform: scale(1);
  }
  50% {
    box-shadow: 0 0 0 8px rgba(239, 68, 68, 0);
    transform: scale(1.05);
  }
}
/* 持续时间: 2s */
/* 缓动: ease-in-out */
```

**规格**：
```
高度: 24px
内边距: 6px 10px
圆角: 9999px
字体: 10px Bold
图标: 12px
```

### 4. 标签页卡片 v2 (TabCardEnhanced)

**外观**：
在原有 TabCard 基础上增加：

#### 4.1 评分指示器 (ScoreIndicator)
```
外观: 卡片右上角半圆形进度条
显示: 0-100 的可关闭指数
颜色: 根据分数渐变(绿→黄→红)
动画:
  - 入场: 从0填充到当前值
  - 分数变化: 平滑过渡
尺寸: 32px × 16px (半圆)
```

#### 4.2 快速关闭热区 (QuickCloseHotspot)
```
外观: 右上角 × 按钮，但更有设计感
设计: 圆形按钮，内有优雅的 × 图标
动画:
  - Hover: 按钮放大，背景变红
  - 点击: × 变圆圈打勾，然后卡片飞出
```

**卡片状态增强**：
```
- 可关闭态: 左侧出现彩色条带(渐变色)
- 今天重要: 顶部出现绿色小圆点
- 悬停态: 整张卡片轻微上浮 + 阴影加深
- 关闭中态: 向右飞出 + 透明度渐变
```

**关闭动画**：
```css
@keyframes tab-close-fly {
  0% {
    transform: scale(1) translateX(0);
    opacity: 1;
  }
  50% {
    transform: scale(1.1) translateX(20px);
    opacity: 0.8;
  }
  100% {
    transform: scale(0.8) translateX(100px) translateY(-50px);
    opacity: 0;
  }
}
/* 持续: 400ms */
/* 缓动: cubic-bezier(0.4, 0, 0.2, 1) */
```

### 5. 批量确认弹窗 (BatchConfirmModal)

**外观**：
- 居中弹出，毛玻璃背景遮罩
- 卡片式弹窗，圆角大
- 顶部：图标 + 统计数字
- 中部：影响预览（显示要关闭的标签列表片段）
- 底部：取消 + 确认按钮

**状态列表**：
```
- 默认态: 入场动画(缩放0.9→1 + 透明度)
- 确认中态: 按钮显示加载动画，列表项依次淡出
- 成功态: 绿色打勾动画，卡片上移淡出
- 失败态: 红色抖动，显示错误信息
```

**内容展示**：
```
统计区:
- 大字显示数量: "即将关闭 7 个标签页"
- 小字说明: "包括 3 个可能不会再看的页面"

预览区:
- 最多显示 5 个标签缩略
- 更多时显示 "+2 个..."
```

### 6. 成就提示 (AchievementToast)

**外观**：
- 类似游戏成就解锁的提示
- 从顶部滑入
- 左侧：成就图标
- 右侧：文案 + 分数加成（如果设计积分系统）

**动画**：
```css
/* 入场 */
@keyframes achievement-in {
  0% {
    transform: translateY(-100%) scale(0.8);
    opacity: 0;
  }
  60% {
    transform: translateY(10%) scale(1.02);
  }
  100% {
    transform: translateY(0) scale(1);
    opacity: 1;
  }
}

/* 光效 */
@keyframes achievement-shine {
  0% { left: -100%; }
  100% { left: 200%; }
}

/* 停留 3s 后滑出 */
@keyframes achievement-out {
  0% {
    transform: translateY(0);
    opacity: 1;
  }
  100% {
    transform: translateY(-100%);
    opacity: 0;
  }
}
```

**文案示例**：
```
🎉 "断舍离大师" - 一次关闭 5 个标签页
📊 "清爽达人" - 今日清理 20 个标签页
⚡ "极速整理" - 10 秒内关闭 10 个
```

## 四、交互规范

### 核心交互流程

#### 流程 1: 发现建议
```
1. 用户打开新标签页
2. 分类标签和统计数字从右侧滑入
3. 如果有可清理项，清理建议卡片从下方弹入
4. 卡片中的数字从 0 滚动到实际值
5. 智能图标开始脉冲动画
```

#### 流程 2: 快速关闭单个
```
1. 用户看到某个 Tab 左上角的"可关闭"标识
2. Hover: 标识轻微放大
3. 点击标识 或 点击右上角 ×
4. Tab 卡片执行关闭动画（向右飞+淡出+粒子）
5. Toast 显示"已关闭" + 小小的成就积分
6. 分类统计数字实时更新
```

#### 流程 3: 一键清理
```
1. 用户点击"一键清理"按钮
2. 弹出确认弹窗，显示影响统计
3. 用户确认
4. 按钮变为进度动画
5. 每个待关闭的卡片依次执行关闭动画（间隔 80ms）
6. 全部完成后，显示成就提示："断舍离大师！"
7. 弹窗关闭，页面刷新统计
```

#### 流程 4: 分类浏览
```
1. 用户点击分类标签（如"随手开的"）
2. 当前内容淡出
3. 标签切换动画
4. 新内容按分类展示，标签页依次入场（stagger 动画）
5. 页面滚动到标签页区域
```

## 五、技术实现要点

### CSS 动画类
```css
/* 脉冲动画 */
.animate-pulse-glow {
  animation: pulse-glow 2s ease-in-out infinite;
}

/* 关闭飞走 */
.animate-close-fly {
  animation: tab-close-fly 0.4s cubic-bezier(0.4, 0, 0.2, 1) forwards;
}

/* 弹入 */
.animate-bounce-in {
  animation: bounce-in 0.5s cubic-bezier(0.68, -0.55, 0.265, 1.55);
}

/* 数字滚动 */
.animate-number-roll {
  animation: number-roll 0.4s ease-out;
}
```

### React 动画建议
- 使用 CSS 动画 + transition
- 列表入场使用 stagger 效果
- 状态变化使用 AnimatePresence 模式

### 性能考虑
- 使用 `transform` 和 `opacity` 做动画（GPU 加速）
- 关闭动画使用 `will-change` 提示
- 避免动画影响布局触发
